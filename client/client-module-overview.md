## Общая модель

Клиентская часть BPMSoft построена на AMD-модулях. Типовой schema module возвращает объект конфигурации:

```javascript
define("MyPage", ["MyPageResources", "ServiceHelper"], function(resources, ServiceHelper) {
    return {
        entitySchemaName: "MyEntity",
        mixins: {},
        messages: {},
        attributes: {},
        details: {},
        modules: {},
        methods: {},
        rules: {},
        diff: []
    };
});
```

Один модуль обычно описывает страницу, секцию, деталь, мини-карточку, utility или provider.

## Основные типы клиентских модулей

| Тип | Базовый компонент | Для чего используется |
| --- | ----------------- | --------------------- |
| Page | `BasePageV2` | карточка записи, сохранение, validation, details |
| Section | `BaseSectionV2` | список записей, filters, actions, dashboards |
| Detail | `BaseDetailV2`, `BaseGridDetailV2` | дочерние записи внутри карточки |
| MiniPage | mini page schema | быстрое создание/просмотр |
| Utility | singleton/helper module | общие функции, service wrappers |
| Provider | `Ext.define` singleton/class | интеграции, CTI, внешние библиотеки |
