# Документация платформы BPMSoft — ядро платформы

Справочная документация по архитектуре, классам и паттернам платформы **BPMSoft** (платформа 1.9). Предназначена для разработчиков, создающих решения на платформе.

## Начало работы

**Главное оглавление:** [INDEX.md](INDEX.md)

В INDEX.md вы найдёте:

- **Getting Started** — первые шаги за 15 минут (структура проекта, Hello World, цикл разработки)
- **Путь обучения** — базовый → продвинутый → экспертный уровень
- **Быстрый старт по задачам** — таблица «задача → документ»
- **Точки входа** — по классу, по ошибке, по паттерну
