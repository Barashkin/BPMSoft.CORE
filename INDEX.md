## Getting Started — первые шаги за 15 минут

### 1. Структура проекта

**Монорепо Base (документы здесь же):**

```text
Base/
├── Autogenerated/Src/          ← автосгенерированный код платформы
│   ├── {Класс}.{Пакет}.cs      ← сервер
│   ├── {Класс}.{Пакет}.js      ← клиент
│   └── {Класс}.{Пакет}.less    ← стили
├── docs/                       ← эта документация
└── …
```

**Приложение конфигурации (корень workspace в Cursor, автокод у корня):**

```text
{ConfigurationRoot}/            ← например Bpm.Configuration
├── Autogenerated/              ← локальный автоген платформы
│   └── Src/ …
├── Src/                        ← код конфигурации
├── Pkg/ , Tests/
└── .docs/ или docs/           ← документацию размещают рядом; пути вида «Autogenerated/Src» считать от этого корня
```

Подробно: [configuration-workspace-layout.md](architecture/configuration-workspace-layout.md).

### 2. Hello World — минимальное рабочее решение

**Создание сущности** → определите схему в `*Schema.Base.cs` (см. [entity-schema-overview.md](server/entity-schema-overview.md))

**Создание WCF-сервиса** → добавьте класс с `[ServiceContract]` (см. [services-overview.md](server/services-overview.md))

**Создание страницы карточки** → создайте AMD-модуль с `entitySchemaName` (см. [pages-sections-details.md](client/pages-sections-details.md))

### 3. Цикл разработки

```text
Редактирование кода → Компиляция пакета → Перезапуск приложения → Тестирование в браузере
```

### 4. Ключевые концепции

| Концепция | Что это | Где изучить |
| ----------- | --------- | ------------- |
| Entity Schema | Описание таблицы БД (колонки, типы, связи) | [entity-schemas.md](server/entity-schemas.md) |
| UserConnection | Контекст текущего пользователя (доступен везде) | [platform-overview.md](architecture/platform-overview.md) |
| ESQ (EntitySchemaQuery) | ORM для запросов к данным | [esq-overview.md](server/esq-overview.md) |
| AMD Module | Единица клиентского кода (define/require) | [modules.md](client/modules.md) |
| diff | Декларативное описание UI-изменений | [modules.md](client/modules.md) |
| Sandbox | Шина сообщений между модулями | [modules.md](client/modules.md) |
