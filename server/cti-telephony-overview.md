# CTI Telephony Overview

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: CTI, telephony, WebRTC, calls -->

> Входная точка в CTI / Telephony Dive: настройки телефонии, провайдеры,
> WebRTC, модель звонков, CTI-панель, идентификация абонента, история,
> записи разговоров, права и диагностика.

## Общая цепочка

```text
CtiBaseHelper
  -> SysSettings.SysMsgLib
  -> SysMsgLib / SysMsgUserSettings
  -> CtiRightsService
  -> CtiProviderInitializer
  -> BaseCtiProvider / BaseWebRTCProvider / provider-specific module
  -> CtiPanel / CommunicationPanel
  -> Call / SysActiveCall
```

Для WebRTC дополнительно используется транспортный слой:

```text
BaseWebRTCProvider
  -> MsgUtilService.svc
  -> WebRTCAdapter
  -> WebRTCSession
  -> JsSipLibrary
```

## Основные контуры

| Контур | Компоненты | Назначение |
| ------ | ---------- | ---------- |
| Initialization | `CtiBaseHelper`, `CtiProviderInitializer` | чтение настроек и загрузка provider |
| Metadata | `SysMsgLib`, `SysMsgUserSettings` | выбранная библиотека и параметры пользователя |
| Rights | `CtiRightsService`, `LicOperations` | проверка operation licenses |
| Providers | `BaseWebRTCProvider`, `UisCtiProvider`, `InfinityCtiProvider` | соединение с телефонией |
| Call model | `Call`, `SysActiveCall`, `VwRecentCall` | запись и активный контекст звонка |
| Client UI | `CtiPanel`, `CommunicationPanel`, `MainHeaderSchema` | управление звонком |
| Identification | `CtiPanelIdentificationUtilities`, `CTISearchResult` | поиск и выбор абонента |
| Records | `TelephonyCallRecordServiceHelper`, `UisCallRecordService` | ссылки и выгрузка записей |
