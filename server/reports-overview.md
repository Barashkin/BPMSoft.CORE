# Reports Printables Overview

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->

> Входная точка в Reports/Printables Dive: движок отчётов, Word/FastReport, печатные формы, шаблоны, async PDF и клиентское меню печати.

## Общая модель

BPMSoft поддерживает несколько контуров отчётов:

| Контур | Компоненты | Назначение |
| ------ | ---------- | ---------- |
| Unified engine | `IReportEngine`, `ReportEngine`, `ReportEngineService` | единый API для Word и FastReport |
| Legacy Word/DevExpress | `ReportService`, `ReportHelper`, `IReportGenerator` | генерация по `SysModuleReport` и скачивание по session key |
| FastReport | `FastReportService`, `ReportGenerator`, data sources | PDF по `.frx` шаблонам |
| Word designer | `WordPrintablePage`, `WordReportingDesignService` | настройка шаблонов и макросов |
| Client print UI | `PrintReportUtilities`, `ReportEngineClient` | меню печати в карточках и секциях |
| Async PDF | `AsyncReportGenerationService`, callback service | долгая генерация и уведомления |
| Metadata | `SysModuleReport`, `StoredReport`, `SysAllowedReportFormat` | регистрация, шаблоны и форматы |
