# Global Search Overview

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: GlobalSearch, Elasticsearch, indexing, search -->

> Входная точка в Search / Global Search Dive: сервис поиска, провайдеры,
> индексация, UI результатов, настройки, SSP-ограничения и диагностика.

## Общая цепочка

```text
GlobalSearchResultPage / GlobalSearchModule
  -> GlobalSearchService.Search
  -> GlobalSearchHelper / GlobalSearchColumnUtils
  -> GsSearchProvider или ESSearchProvider
  -> Global Search service / Elasticsearch
  -> BpmSearchResponse
```

Индексация идет отдельным контуром:

```text
ConfigurationBuildWatcher / GlobalSearchEventListener
  -> IndexingConfigService
  -> IndexingConfigSender
  -> LogstashConfigsJob / BaseIndexer
  -> indexing service
```

## Основные контуры

| Контур | Компоненты | Назначение |
| ------ | ---------- | ---------- |
| Service | `GlobalSearchService` | proxy endpoint поиска |
| Helper | `GlobalSearchHelper` | права, подсветка, результат |
| Providers | `GsSearchProvider`, `ESSearchProvider` | новый GS service или ES |
| Query | `ESQueryBuilder`, `GsSearchQuery` | построение запросов |
| Indexing | `IndexingConfigService`, `BaseIndexer` | конфиги и real-time indexing |
| UI | `GlobalSearchResultPage`, row schemas | страница результатов |
| Settings | `GlobalSearchUrl`, `GlobalSearchIndexedDataConfig` | runtime-настройки |
| SSP | `GlobalSearchSSPHelper` | portal ACL и allowed columns |
