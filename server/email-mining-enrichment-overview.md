# Email Mining Enrichment Overview

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: EmailMining, Enrichment, mailbox, Activity -->

> Входная точка в Email Mining / Enrichment Dive: синхронизация писем,
> извлечение сущностей из тела письма, обогащение контактов и аккаунтов,
> фоновые jobs, UI-actions и диагностика.

## Границы

В `Autogenerated/Src` представлены:

- mailbox settings и jobs для IMAP/Exchange/mail resync;
- `EmailMiningJob`, `EmailMiner`, `EmailMiningServiceProxy`;
- модели `EnrchEmailData`, `EnrchTextEntity`, found/rejected/processed data;
- searchers для Contact/Account и dedup mined data;
- `EnrichContactService` и клиентский enrichment UI;
- общий `EnrichmentService` для account enrichment.

Вне этого среза:

- низкоуровневые IMAP/SMTP/Exchange клиенты;
- cloud NER/text mining engine;
- реализации `BPMSoft.Enrichment.Interfaces`;
- детальный BPMN flow generated process-схем.

## Общий поток

```text
Mailbox sync / Activity email
  -> Activity.EnrichStatus = Inactive
  -> EmailMiningJob
  -> EmailMiner
  -> EmailMiningServiceProxy.getEmailEntitiesBatch
  -> EnrchEmailData + EnrchTextEntity
  -> ContactSearcher / AccountSearcher / Deduplicator
  -> EnrichContactService + EmailEnrichmentMixin UI
```
