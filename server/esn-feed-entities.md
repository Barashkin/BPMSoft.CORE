# ESN Feed Entities

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: ESN, SocialMessage, SocialLike, SocialSubscription -->

> ESN-модель строится вокруг `SocialMessage`: это и пост, и комментарий.
> Лайки, подписки, каналы и уведомления хранятся отдельными сущностями.

## SocialMessage

Файл: `Autogenerated/Src/SocialMessageSchema.ESN.cs`.

`SocialMessage` наследуется от `BaseEntitySchema`. Primary display column —
`Message`.

Ключевые колонки:

| Колонка | Назначение |
| ------- | ---------- |
| `Message` | текст поста или комментария |
| `Parent` | самоссылка; если заполнена, запись является комментарием |
| `EntitySchemaUId` | схема бизнес-записи, к которой привязан пост |
| `EntityId` | идентификатор бизнес-записи |
| `LikeCount` | счетчик лайков |
| `CommentCount` | счетчик комментариев |
| `LastActionOn` | дата последнего действия |

`Parent` индексирован и cascade, поэтому удаление родительского сообщения
затрагивает дочерние комментарии.

## SocialMessageEntity

Файл: `Autogenerated/Src/SocialMessageEntitySchema.ESN.cs`.

Связывает пост с произвольными бизнес-записями:

```text
SocialMessage -> EntitySchemaUId + EntityId
```

Это используется для тегов/упоминаний сущностей и дополнительных связей
сообщения с объектами.

## SocialLike

Файл: `Autogenerated/Src/SocialLikeSchema.ESN.cs`.

Лайк — отдельная запись:

| Колонка | Назначение |
| ------- | ---------- |
| `User` | `SysAdminUnit` пользователя |
| `SocialMessage` | сообщение, которое лайкнули |

Связь с `SocialMessage` cascade, поэтому лайки удаляются вместе с сообщением.

## SocialSubscription

Файл: `Autogenerated/Src/SocialSubscriptionSchema.ESN.cs`.

Подписка пользователя на произвольную сущность:

| Колонка | Назначение |
| ------- | ---------- |
| `SysAdminUnit` | пользователь/роль |
| `EntitySchemaUId` | схема подписки |
| `EntityId` | запись подписки |
| `CanUnsubscribe` | можно ли отписаться |
| `RightsChangedOn` | дата изменения прав |

Схожий контур:

- `SocialUnsubscription` — исключения/отписки;
- `VwSocialSubscription` — view для отображения;
- `VwSocialUnsubscription` — view для отписок.

## SocialChannel

Файл: `Autogenerated/Src/SocialChannelSchema.ESN.cs`.

Канал ленты хранит название, изображение и `PublisherRightKind`, который
определяет, кто может публиковать в канал.

Связанные helper schemas:

- `SocialChannelFolder`;
- `SocialChannelInFolder`;
- `SocialChannelTag`;
- `SocialChannelInTag`;
- `SocialChannelPublisher`.

## События и generated process

Схемы ESN создают embedded events processes:

- `SocialMessage_ESNEventsProcess`;
- `SocialLike_ESNEventsProcess`;
- `SocialSubscription_ESNEventsProcess`.

Бизнес-логика listeners и `EsnService` частично находится за границей
`Autogenerated/Src`, но автогенерат показывает точки расширения и модель.
