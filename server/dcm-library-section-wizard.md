# DCM Library, Section Settings And Wizard

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: DcmLibrary, SectionDcmSettings, SectionWizardCasesSettings -->

> DCM library и section wizard отвечают за список DCM-схем, включение/выключение
> схем, выбор stage column и сохранение DCM-настроек секции.

## VwDcmLibSection

Файл: `Autogenerated/Src/VwDcmLibSection.DcmLibrary.js`.

Секция:

- `entitySchemaName = "VwSysDcmLib"`;
- использует `DcmSchemaManager`;
- показывает DCM-схемы библиотеки;
- обновляет display value для filters и stage column;
- включает/выключает DCM-схемы через `DcmSchemaManager.setEnabled`;
- в wizard mode публикует `EnabledDcmSchemas`.

Ключевые attributes:

| Attribute | Роль |
| --------- | ---- |
| `EntitySchemaUId` | schema секции |
| `PackageUId` | пакет |
| `IsSelectedCaseActive` | активность выбранной DCM-схемы |
| `ActiveProcessFilterColumnName` | колонка active flag |
| `StageEntitySchema` | schema для stage display values |

## SectionDcmSettings

Файл: `Autogenerated/Src/SectionDcmSettings.DcmLibrary.js`.

Модуль управляет настройками секции:

- `SysModuleEntityId`;
- `SspSysModuleEntityId`;
- `PackageUId`;
- `SysDcmSettingsItem`;
- `StageColumn`;
- `FilterColumn`;
- `Filters`;
- `StageColumnsList`;
- `DcmSchemaList`;
- `FilterColumnsList`.

При reload он переинициализирует:

- `DcmSchemaManager`;
- `SysDcmSettingsManager`;
- `SysDcmSchemaInSettingsManager`;
- settings item;
- section entity schema;
- column values.

## SectionWizardCasesSettings

Файл: `Autogenerated/Src/SectionWizardCasesSettings.DcmLibrary.js`.

Wizard step встраивает два модуля:

| Module | Schema |
| ------ | ------ |
| `SectionDcmSettingsModule` | `SectionDcmSettings` |
| `VwDcmLibSectionModule` | `VwDcmLibSection` через `SectionModuleV2` |

Атрибут `Initialized` зависит от:

- `SectionDcmSettingsInitialized`;
- `SectionDcmLibraryInitialized`.

## ServerChannel reload

`SectionWizardCasesSettings` слушает `BPMSoft.ServerChannel` и реагирует на:

```text
ReloadSectionWizardCaseSettings
```

Если `dcmSettingsId` совпадает с текущими настройками, wizard публикует:

- `ReloadCaseSettings`;
- `ReloadDcmLibGridData`.

## Включение DCM-схемы

`VwDcmLibSection.setEnabledDcmSchema`:

1. Берёт active row.
2. Формирует `{ items: [schemaUId], enabled }`.
3. Вызывает `BPMSoft.DcmSchemaManager.setEnabled`.
4. Перезагружает grid.
5. В wizard mode публикует `EnabledDcmSchemas`.

Если отключается последняя активная схема, показывается confirmation dialog.

## Практические правила

- Если DCM-схема не видна в wizard, проверяйте `VwSysDcmLib` и связь с settings.
- Если stage column отображается как многоточие, проверьте загрузку `StageEntitySchema`.
- Если wizard не снимает mask, проверьте оба initialized-сообщения.
- Если настройки не обновились после сохранения дизайнера, проверьте
  `ReloadSectionWizardCaseSettings`.
