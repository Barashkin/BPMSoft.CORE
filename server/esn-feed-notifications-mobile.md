# ESN Feed Notifications And Mobile

<!-- Версия: 1.1 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: ESN, notifications, mobile, portal, Reminding -->

> Уведомления ESN хранятся отдельными записями, а мобильный клиент имеет свой
> набор list/grid/edit modules поверх той же модели `SocialMessage`.

## ESNNotification

Файл: `Autogenerated/Src/ESNNotificationSchema.ESN.cs`.

Ключевые колонки:

| Колонка | Назначение |
| ------- | ---------- |
| `Type` | тип уведомления |
| `IsRead` | прочитано ли уведомление |
| `Owner` | контакт получателя |
| `SocialMessage` | связанное сообщение |

`Owner` и `SocialMessage` имеют cascade. Удаление связанного сообщения удаляет
уведомления.

## Граница с Reminding

ESN notifications — отдельная подсистема поверх `ESNNotification` и
`SocialMessage`. NUI notification center использует `Reminding`,
`RemindingJob`, `NotificationSender` и handlers доставки.

Общие counters могут учитывать ESN через `ESNNotificationCounter`, но
диагностику feed/comment/mention уведомлений начинайте в ESN-контуре, а не с
`Reminding`.

## Notification types and settings

Связанные схемы:

- `ESNNotificationTypeSchema.ESN.cs`;
- `EsnNotificationSettingsSchema.ESN.cs`.

`EsnNotificationSettings` хранит персональные настройки, например запрет
уведомлений о новых сообщениях.

## Client notification module

Файлы:

- `ESNNotificationModule.ESN.js`;
- `ESNNotificationModule.ESN.less`;
- `ESNNotificationSchema.ESN.js`;
- `ESNNotificationSchemaCSS.ESN.less`.

Клиентский контур показывает уведомления и работает с channel header из
`ESNConstants`.

## ESNConstants

Файл: `Autogenerated/Src/ESNConstants.ESN.js`.

Содержит:

- UId системных ESN-сущностей;
- заголовки websocket/notification channel;
- коллекцию схем с primary image column;
- shared constants для feed UI.

## Mobile feed

Ключевые файлы:

| Файл | Назначение |
| ---- | ---------- |
| `MobileFeedList.ESN.js` | список feed items, форматирование HTML и ссылок |
| `MobileFeedCommentsList.ESN.js` | список комментариев |
| `MobileSocialMessageGridPage.ESN.js` | grid page сообщений |
| `MobileSocialMessageGridPageController.ESN.js` | controller grid page |
| `MobileSocialMessageGridPageStore.ESN.js` | store grid page |
| `MobileSocialMessageEditPage.ESN.js` | редактирование сообщения |
| `MobileSocialMessagePreviewPage.ESN.js` | preview page |
| `MobileSocialMessageLikeManager.ESN.js` | likes на mobile |
| `MobileSocialMessageEntityLoader.ESN.js` | загрузка связанных сущностей |
| `MobileSocialMessageUtilities.ESN.js` | общие mobile utilities |

`MobileFeedList` дополнительно чистит HTML сообщения: оставляет `a` и `br`,
распознает URL/email, формирует mobile links и обрезает HTML без разрыва тегов.

## Portal notes

SSP/Portal влияет на ESN в нескольких местах:

- `ESNFeedModuleService` добавляет портал-контакты в mentions;
- feature `IsSSPContactSocialMentions` расширяет подбор contacts;
- `SocialMentionSchema.SSP.cs` расширяет mention schema;
- `SubscribersDetailV2.ESN.js` использует portal role filters.
