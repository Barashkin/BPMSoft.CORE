# ESN Feed Social Overview

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: ESN, SocialMessage, feed, timeline, social -->

> Входная точка в ESN / Feed / Social Dive: внутренняя лента, посты,
> комментарии, лайки, подписки, mentions, уведомления, timeline, mobile и
> граница с внешними social network integrations.

## Границы

Внутренний ESN:

- `SocialMessage`, `SocialLike`, `SocialSubscription`;
- `SocialChannel`;
- `ESNFeedModule`, `SocialFeed`, `SocialFeedUtilities`;
- `EsnCenter`, `ESNFeedModuleService`;
- `ESNNotification`;
- mobile ESN modules.

Внешние соцсети:

- `SocialSearch.SocialNetworkIntegration.js`;
- `BaseSocialPage.SocialNetworkIntegration.js`;
- `GoogleClientConnector.SocialNetworkIntegration.js`;
- `SocialCommunicationDetail.SocialNetworkIntegration.js`.

OCC/чат и Sender не относятся к ESN: они описаны в OCC-документации.

## Общая цепочка

```text
SocialFeed / ESNFeedModule / Timeline
  -> EsnService / ESNFeedModuleService
  -> EsnCenter
  -> IEsnMessageReader / IEsnMessageRedactor / IEsnLikeRepository
  -> SocialMessage / SocialLike / ESNNotification
```

Mentions:

```text
SocialMentionUtilities
  -> ESNFeedModuleService
  -> ContactForMention
  -> SocialMentionSearchRule
```
