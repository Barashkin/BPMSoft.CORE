# Administration And Configuration Overview

<!-- Версия: 1.1 | Обновлено: 2026-04-28 | Платформа: BPMSoft 1.9 -->
<!-- Теги: Administration, Configuration, SysAdminUnit, SysSettings -->

> Входная точка в Administration / Configuration Dive: админка,
> оргструктура, настройки, справочники, рабочие места, пакеты и feature flags.

## Общая модель

| Контур | Компоненты | Назначение |
| ------ | ---------- | ---------- |
| System Designer | `SystemDesigner`, designer sections | плитки администрирования |
| Users/roles | `SysAdminUnit`, `SysAdminUnitInRole` | оргструктура |
| SysSettings | `SysSettings`, `SysSettingsValue` | системные настройки |
| Lookups | `SysLookup`, `LookupManager` | справочники |
| Workplaces | `SysWorkplace`, `SysModuleInWorkplace` | рабочие места |
| Sections | `SysModule`, `ConfigurationSectionHelper` | навигация и секции |
| Packages | `SysPackage`, dependencies | конфигурация и пакеты |
| Services | `AdministrationService`, partial services | admin API |
| Features | `FeatureService`, `AdminUnitFeatureState` | feature states |
