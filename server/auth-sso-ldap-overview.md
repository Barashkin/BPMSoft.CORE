# Auth SSO LDAP Overview

<!-- Версия: 1.1 | Обновлено: 2026-04-28 | Платформа: BPMSoft 1.9 -->
<!-- Теги: auth, SSO, LDAP, OAuth, SSP -->

> Входная точка в Auth / SSO / LDAP Dive: LDAP-синхронизация, OAuth 2.0,
> SAML/OIDC-границы, SSP-регистрация, TOTP-сброс пароля, сессии и
> административные операции.

## Границы

В `Autogenerated/Src` хорошо представлены:

- LDAP-настройки, connector, retry/rebind, sync staging tables и процессы;
- OAuth 2.0 client apps, resources, scopes, token storage и UI;
- SSP self-registration, password recovery, TOTP recovery link;
- `SysUserSession` и admin endpoints для завершения сессий;
- SAML field mapping и `SamlResponse` schema.

В этом срезе почти не представлены:

- полноценная NUI login page;
- низкоуровневые cookies/forms authentication;
- end-to-end SAML/OIDC redirect/assertion pipeline.

Эти части обычно находятся в runtime/platform host или других пакетах.

## Основные контуры

LDAP:

```text
LDAPServerSettings UI
  -> LDAPSysSettingsService
  -> LdapSettingsProvider
  -> LdapConnector / LdapUtilities
  -> SysLDAPSynchUser / SysLDAPSynchGroup / LDAPElement
  -> SysAdminUnit / SysUserInRole
```

OAuth 2.0:

```text
OAuthConfigService / OAuth20AppPage
  -> OAuthClientApp / OAuthResource / OAuthResourceInClient
  -> OAuthApplications / OAuthAppScope / OAuthTokenStorage
  -> OAuthClientAppListener / ServiceOAuthAuthenticatorEndpoint
  -> IdentityServiceWrapper / external OAuth provider
```

SSP registration:

```text
UserManagementModule
  -> UserManagementService / TotpSendResetPasswordLinkService
  -> RegistrationHelper
  -> Contact / SysAdminUnit / EmailTemplate
```
