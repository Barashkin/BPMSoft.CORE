# SSP Portal Overview

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: SSP, Portal, portal users, access list -->

> Входная точка в SSP / Portal Dive: пользователи портала, доступные
> схемы/колонки, маршруты сервисов, portal sections, главная страница и
> self-service сценарии.

## Общая модель

| Контур | Компоненты | Назначение |
| ------ | ---------- | ---------- |
| Portal users | `SysAdminUnit`, `VwSSPSysAdminUnit` | пользователи с `ConnectionType = SSP` |
| User management | `SspUserManagementService` | создание и приглашение portal users |
| Admin API | `AdministrationServicePortalUsers` | CRUD portal users из админки |
| Schema ACL | `SysSSPEntitySchemaAccessList`, `PortalSchemaAccessList` | разрешенные схемы |
| Column ACL | `PortalColumnAccessList` | разрешенные колонки |
| Portal sections | `SysModuleEntityInPortal`, `PortalSectionHelper` | публикация секций в портал |
| Main page | `PortalMainPageBuilder`, `PortalMainPageModule` | главная страница портала |
| Self-service | `UserManagementService`, `RegistrationHelper` | регистрация и сброс пароля |
| External access | `ExternalAccess*`, `TempAccess*` | изолированный/временный доступ |
