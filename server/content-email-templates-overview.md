# ContentBuilder And Email Templates Overview

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: ContentBuilder, EmailTemplate, MJML, macros, multilingual -->

> Входная точка в ContentBuilder / Email Templates Dive: хранение шаблонов,
> визуальный редактор, экспорт HTML, мультиязычность, макросы и отправка.

## Общая модель

| Контур | Компоненты | Назначение |
| ------ | ---------- | ---------- |
| Storage | `EmailTemplate`, `EmailTemplateLang` | шаблон и переводы |
| ContentBuilder | `EmailTemplateContentBuilder` | визуальное редактирование |
| Export | `ContentExporterFactory`, `MjmlContentExporter` | config to HTML |
| Preview | `EmailTemplatePageV2`, iframe | просмотр body |
| MLang | `MultiLanguageEmailContentBuilder` | языковые вкладки |
| Macros | `EmailTemplateUserTaskMacrosHelper` | подстановка значений |
| Process | `EmailTemplateUserTask` | отправка из процесса |
| Sending | `EmailSendService`, `ActivityEmailSender` | отправка Activity email |
