## Основные потоки

### Настройка ящика

```text
Mailbox settings UI
  -> MailboxSynchronizationSettingsService / MailboxSettingsService
  -> MailboxSyncSettings
  -> MailServer / OAuthTokenStorage
  -> create/remove sync jobs
```

### Плановая синхронизация

```text
MailboxSettingsService.CreateDeleteSyncJob
  -> ISyncJobScheduler.CreateSyncJob
  -> LoadExchangeEmailsProcess / SyncExchangeActivitiesProcess / SyncExchangeContactsProcess
  -> ExchangeUtility / sync providers
  -> Activity / Contact changes
```

### Listener-события

```text
Exchange listener
  -> ExchangeListenerService.NewEmail / ProcessFullEmail / ProcessMailboxState
  -> IExchangeEventsProcessor / IEmailEventsProcessor / IMailboxStateEventsProcessor
  -> Activity and mailbox state updates
```

### Calendar / meeting invitations

```text
Activity page mixins
  -> MeetingService
  -> IMeetingService
  -> invitations, change checks, calendar sync metadata
```
