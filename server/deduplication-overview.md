# Deduplication Overview

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->

> Входная точка в Deduplication Dive: правила дублей, поиск, массовая дедупликация, merge, UI и диагностика.

## Общая модель

Deduplication в BPMSoft состоит из нескольких контуров:

| Контур | Компоненты | Назначение |
| ------ | ---------- | ---------- |
| Rules | `DuplicatesRule`, rule body, filters | какие поля сравнивать |
| On-save search | `DeduplicationService.FindDuplicatesOnSave` | поиск похожих записей при сохранении |
| Result processing | `DeduplicationProcessing` | groups, rowConfig, rights-aware output |
| Merge | `DeduplicationMergeHandler` | золотая запись, перенос ссылок, история |
| Bulk search | `BulkDeduplicationManager`, external WebApi | массовый поиск дублей |
| Scheduler/jobs | `CheckDeduplicationTaskStatusJobExecutor` | опрос статуса bulk-задачи |
| Client UI | widgets, pages, merge module | карточки, секции, страницы дублей |
| Storage/logs | duplicate tables, history, exec log | результаты, блокировки, журнал |
