# Reports FastReport

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: FastReport, FastReportService, FastReportTemplate, data source, PDF -->

> FastReport-печатные формы: `.frx` шаблоны, источники данных, генерация PDF.

## Компоненты

| Компонент | Назначение |
| --------- | ---------- |
| `FastReportTemplate` | хранит `.frx` template в поле `Data` |
| `FastReportDataSource` | описывает источник данных |
| `FastReportTemplateProvider` | загружает шаблон |
| `FastReportDataSourceBuilderResolver` | подбирает builders источников |
| `ReportGenerator` | генерирует PDF |
| `FastReportService` | WCF endpoint для legacy FastReport API |

## Генерация

`FastReportService` создаёт `ReportGenerator` из template provider и data source resolver.

```csharp
IReportTemplateProvider reportTemplateProvider = GetReportTemplateProvider();
IDataSourceBuilderResolver dataSourceBuilderResolver = GetDataSourceBuilderResolver();
return new ReportGenerator(reportTemplateProvider, dataSourceBuilderResolver);
```

В unified engine FastReport вызывается через abstraction `BPMSoft.Reporting.Abstractions.IReportGenerator`.

## Data sources

FastReport может получать данные из:

- ESQ data source;
- custom data source provider;
- нескольких data sources, применённых к одному шаблону.

Для ESQ-фильтров engine передаёт параметр `EsqFilters`.

## Template editing

`FastReportService` умеет загрузить, очистить и сохранить XML шаблон:

- `GetReportTemplate`;
- `ApplyDataSources`;
- `RemoveReportDataSources`;
- `SaveReportTemplate`.

Перед чтением XML используется безопасная настройка `XmlReaderSettings` с запретом DTD.

## Download flow

Legacy `FastReportService` кладёт bytes отчёта в `UserConnection.SessionData` под `ReportCacheKey_*`, а затем `GetReportFile/{key}` отдаёт поток и удаляет cache entry.

## Что не найдено

Отдельный `FastReportViewer` в `Autogenerated/Src` не найден. В этой базе видны service, template/data-source model и filter pages.

## Практические правила

- Для нового server API предпочитайте `IReportEngine`, даже если тип FastReport.
- Для кастомного источника данных реализуйте provider и регистрируйте data source metadata.
- Шаблон `.frx` храните в `FastReportTemplate.Data`.
- Не оставляйте data source definitions в template, если они должны применяться динамически.
- FastReport output в текущей модели — PDF.
