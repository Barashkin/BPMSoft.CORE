# Knowledge Base And Case Terms Overview

<!-- Версия: 1.1 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: KnowledgeBase, SSP, CaseTerm, DCM, Service -->

> Входная точка в Knowledge Base / Case Terms Dive для Base-пакета.
> Здесь документируются фактические компоненты `Autogenerated/Src`: база
> знаний, портал/SSP, строки поиска, DCM case settings и календарный расчёт
> сроков. Полноценные ITSM `Case`, `ServiceItem`, `ServicePact`, SLA и
> routing/escalation в этом Base автогенерате не представлены.

## Границы пакета

В `Base/Autogenerated/Src` есть:

- `KnowledgeBase` entity и UI;
- portal/SSP Knowledge Base;
- Knowledge Base search module;
- Global Search row schemas для `KnowledgeBase` и `Case`;
- DCM section wizard case settings;
- календарные `CaseTerm*` стратегии и logging;
- SSP access primitives, уже описанные в [ssp-portal-overview.md](ssp-portal-overview.md).

В этом пакете не найдено:

- `CaseSchema` как полноценная ITSM-сущность;
- `ServiceItem`, `ServicePact`, `ServiceInServicePact`;
- SLA policy/routing/escalation schemas;
- portal case creation pages.

## Основная карта

```text
KnowledgeBaseSchema
  -> KnowledgeBasePageV2 / KnowledgeBaseSectionV2
  -> KnowledgeBaseFile / KnowledgeBaseTagV2
  -> Global Search row
  -> SSP section/page/search modules
```

Календарные сроки:

```text
DeadlineCalcSchemas
  -> CaseTermStrategyQueue
  -> CaseTermStrategyManager
  -> CaseTermIntervalSelector
  -> CaseTermInterval
  -> TermCalculationLogStore / TermCalculationLogger
```
