# Analytics Dashboards Overview

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: AnalyticsService, dashboards, widgets, metrics -->

> Входная точка в Analytics / Dashboards / Metrics Dive: дашборды,
> виджеты, аналитический сервис, drill-down, секционная аналитика и метрики.

## Общая цепочка

```text
SysDashboard / SysWidgetDashboard
  -> Items + ViewConfig JSON
  -> AnalyticsService / AnalyticsServiceUtils
  -> DashboardItemDataFactory
  -> BaseDashboardItemData / Chart / Indicator / Grid
  -> DashboardEnums + widget modules
  -> DashboardsModule / DashboardBuilder
```

## Основные контуры

| Контур | Компоненты | Назначение |
| ------ | ---------- | ---------- |
| Storage | `SysDashboard`, `SysWidgetDashboard` | хранение dashboard metadata |
| JSON config | `Items`, `ViewConfig` | состав и раскладка виджетов |
| Service | `AnalyticsService` | WCF API данных |
| Utils | `AnalyticsServiceUtils` | чтение JSON и data dispatch |
| Factory | `DashboardItemDataFactory` | выбор server data class |
| Widget registry | `DashboardEnums` | view/design modules |
| Shell | `DashboardsModule`, `DashboardBuilder` | UI страницы дашбордов |
| Drill-down | `ChartDrillDownProvider` | история и детализация графиков |
| Section analytics | `SectionDashboardsModule` | аналитика внутри секций |
| Mobile | `MobileAnalyticsService`, mobile dashboard modules | mobile dashboard runtime |
