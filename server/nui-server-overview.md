# NUI Server Overview

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: NUI, server, services, approval, grid, export -->

> Входная точка в NUI server layer: серверные сервисы и helper-пайплайны,
> которые обслуживают classic UI/NUI сценарии: согласования, операции над
> записями, Excel export, audit archive, конфигурацию секций и рабочих мест.

## Что относится к NUI server layer

В этом dive NUI server layer означает не весь клиентский NUI и не все WCF
сервисы платформы, а серверные классы пакета `NUI`, которые выступают
предметной прослойкой между UI и платформенными API.

Типовые признаки:

- файл `Autogenerated/Src/*.NUI.cs`;
- WCF `[ServiceContract]` + `BaseService`;
- DTO из `BPMSoft.Nui.ServiceModel.DataContract`;
- сценарий вызывается из client module, section/grid/page или portal UI;
- логика тонко оборачивает ESQ, права, AppScheduler, OperationLog или
  configuration helpers.

## Типовой flow

```text
Client module / section / grid
  -> ServiceHelper / AjaxProvider
  -> NUI WCF service
  -> platform API: ESQ, rights, AppScheduler, OperationLog, repository
  -> DTO / ConfigurationServiceResponse / serialized SelectQueryResponse
  -> client UI update
```
