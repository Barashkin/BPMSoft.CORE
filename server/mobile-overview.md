# Mobile Overview

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: Mobile, manifest, sync, offline, push, workplace -->

> Входная точка в Mobile Dive: мобильные манифесты, SDK-страницы, offline sync,
> сервисы, push-уведомления и рабочие места.

## Общая модель

Мобильный слой BPMSoft описывается декларативно и исполняется мобильным SDK.

| Контур | Компоненты | Назначение |
| ------ | ---------- | ---------- |
| Manifest | `MobileApplication*Manifest*.js` | состав приложения |
| Features | `Features`, `CustomSchemas` | условные возможности |
| Sync | `SyncOptions` | offline import/export |
| Pages | `BPMSoft.sdk.GridPage`, `RecordPage` | UI моделей |
| Cache | `ModelCache`, cache managers | локальное хранилище |
| Services | `MobileServiceHelper` | вызовы `rest/{service}/{method}` |
| Push | `MobilePushNotificationReceiver` | открытие записей из push |
| Workplaces | `SysMobileWorkplace` | доступные mobile workplace |
