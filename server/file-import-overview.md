# File Import Overview

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->

> Входная точка в FileImport/Data Import Dive: импорт Excel-файлов, wizard, маппинг, чанки, фоновые процессы и диагностика.

## Общая модель

FileImport в BPMSoft — отдельная подсистема импорта данных из файла. Она не равна обычному File Storage: файл используется как источник данных, а не как вложение записи.

| Слой | Компоненты | Назначение |
| ---- | ---------- | ---------- |
| Client wizard | `FileImportWizard`, step pages | пошаговый UI импорта |
| Upload | `ConfigurationFileApi`, `FileImportUploadFileService` | загрузка файла импорта |
| Service API | `FileImportService`, `FileImportTemplateService` | параметры сессии, mapping, запуск |
| Import state | `ImportSession`, `FileImportParameters`, chunks | состояние и данные импорта |
| Processing | `FileImportProcess`, chunk processors | сохранение сущностей и tags |
| Background recovery | `FileImportBackgroundProcessor` | перезапуск незавершённого persistent import |
| Progress | `ServerChannel`, `FileImportProcessingPage` | обновление процента и переход к результату |
| Result/log | `FileImportResultPage`, `ExcelImportLog` | итоговые счётчики и ошибки |

## Важные границы

- В просмотренном `Autogenerated/Src` основной формат импорта — Excel/OpenXml; отдельный CSV-контур FileImport не найден.
- `FileImportUploadFileService` и `FileUploader` — специализированный upload path для импорта, не обычный `FileRepository` attachment flow.
- `UsePersistentFileImport` меняет хранение состояния и включает recovery незавершённых импортов.
- `CanImportFromExcel` — ключевая operation right для запуска импорта.
