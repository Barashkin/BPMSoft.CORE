> Каталог реальных паттернов отчётов и печатных форм из `Autogenerated/Src`.

## Unified engine

| Паттерн | Файл | Что смотреть |
| ------- | ---- | ------------ |
| Report contract | `IReportEngine.Base.cs` | `ReportSettings`, `IReportResult`, `IReportTemplateInfo` |
| Engine implementation | `ReportEngine.Reports.cs` | выбор Word/FastReport, filters, `UseAdminRights = false` |
| Service facade | `ReportEngineService.Reports.cs` | `GetReportTemplates` DTO |

## Legacy service

| Паттерн | Файл | Что смотреть |
| ------- | ---- | ------------ |
| Create report | `ReportService.NUI.cs` | `CreateReport`, `CreateReportsList` |
| Download by key | `ReportService.NUI.cs`, `ReportHelper.NUI.cs` | `GetReportFile`, session key |
| Format validation | `ReportService.NUI.cs` | `ValidateReportFormat`, `SysAllowedReportFormat` |
| Export right | `ReportHelper.NUI.cs` | `CheckCanExecuteOperation("CanExportGrid")` |

## Word reports

| Паттерн | Файл | Что смотреть |
