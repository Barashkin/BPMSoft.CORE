# Activity Email Overview

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: Activity, Email, ActivityParticipant, EmailSender, mailbox -->

> Входная точка в Activity/Email Dive: активности, письма, участники, вложения, отправка, процессы и синхронизация.

## Общая модель

`Activity` — универсальная сущность для задач, звонков, встреч и email. Конкретный сценарий задаётся сочетанием `Type`, `ActivityCategory`, `Status`, `MessageType` и `EmailSendStatus`.

| Слой | Компоненты | Назначение |
| ---- | ---------- | ---------- |
| Activity model | `ActivitySchema`, `ActivityConsts` | поля, типы, статусы, категории |
| Runtime lifecycle | `Activity.Base.cs` | preview, duration, reminders, participants |
| Participants | `ActivityParticipant`, `EmailParticipantHelper` | To/CC/BCC/From, приглашения, права |
| Files | `ActivityFile`, `EmailFileDetailV2` | вложения email и активностей |
| Sending | `IEmailSender`, `EmailSender`, `ActivityEmailSender` | формирование и отправка письма |
| Processes | `EmailTemplateUserTask`, `SendEmailUserTask`, `ActivityUserTask` | создание и отправка из BPM |
| Sync | `MailboxSyncSettings`, Exchange schemas | почтовые ящики и синхронизация |
| Client UI | `EmailPageV2`, scheduler modules | карточка письма, календарь, HTML body |
