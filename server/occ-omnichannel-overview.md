## Контур OCC

OCC в Base распределён между пакетами:

| Пакет | Роль |
| ----- | ---- |
| `BPMSoftOCC` | ядро чатов: каналы, клиенты, сообщения, операторы, routing, WCF |
| `BPMSoftOCCWAMfmsJson` | WhatsApp MFMS/Edna расширение |
| `BPMSoftSender` | массовые рассылки поверх OCC-каналов |

Высокоуровневая архитектура уже описана в
[bpmsoft-occ.md](../architecture/bpmsoft-occ.md).

## Основные потоки

### Входящее сообщение

```text
Connector
  -> BPMSoftOCCChatRequestService
  -> BPMSoftOCCChatRequest
  -> RequestHandler
  -> BPMSoftOCCClient / BPMSoftOCCChat / BPMSoftOCCChatMessage
  -> MsgChannel "BPMSoftOCCChat"
  -> CommunicationPanel
```

### Исходящее сообщение

```text
CommunicationPanel / API
  -> BPMSoftOCCChatService
  -> ApiService.SendMessage
  -> connector API
  -> MessageStatus callback
  -> BPMSoftOCCChatMessageStatusRequestHandler
  -> MsgChannel "BPMSoftOCCChat"
```

### Sender-рассылка

```text
BPMSoftOCCDelivery
  -> BSDeliveryService / BSSchedulerService
  -> BPMSoftSenderDeliverySchedulerJob
  -> BSDeliveryStrategy
  -> OCC channels / chats / messages
```
