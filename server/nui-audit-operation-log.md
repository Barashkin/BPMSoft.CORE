# NUI Audit Operation Log

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: NUI, AuditService, OperationLog, SysOperationAudit -->

> Серверный audit-контур NUI: архивация журнала операций и связь с
> `SysOperationAudit`, Excel export и административными правами.

## AuditService

Опорный файл: `Autogenerated/Src/AuditService.NUI.cs`.

Сервис минимальный:

```text
AuditService.MoveToArchive(startDate, endDate, operationTypes)
  -> OperationLogUtilities.MoveToArchive(...)
  -> int rowCount
```

Метод принимает даты строками и массив operation types. Если дата не парсится,
возвращает `0`.

## Operation types

`operationTypes` преобразуются в enum `OperationType`. В архив передаются
только значения, которые:

1. успешно парсятся;
2. определены в enum.

Некорректные значения не добавляются в массив типов.

## Что делает MoveToArchive

`AuditService` не реализует архивирование самостоятельно. Он является thin WCF
facade над:

```text
BPMSoft.Core.OperationLog.OperationLogUtilities.MoveToArchive
```

Вся низкоуровневая логика перемещения записей находится в core, не в
`Autogenerated/Src`.

## Связь с Excel export

`ExportToExcelService` может писать записи в `SysOperationAudit`, если включена
настройка:

```text
UseAdminOperationImportExport
```

Это не тот же flow, что архивирование, но оба сценария относятся к
операционному аудиту:

- export создаёт audit запись;
- `AuditService.MoveToArchive` архивирует operation log за период и типы.

## Граница с Administration и Security

Administration отвечает за UI и настройки управления журналом. Security
отвечает за operation rights вроде `CanManageChangeLog` и доступ к
административным действиям.

NUI audit document фиксирует только server endpoint и его контракт.

## Типовые сбои

| Симптом | Возможная причина |
| ------- | ----------------- |
| Возвращается `0` | `startDate` или `endDate` не распарсились |
| Архив пустой | operation types не совпали с enum |
| Нет доступа из UI | operation right / admin permission |
| Экспорт не пишет audit | выключен `UseAdminOperationImportExport` |
