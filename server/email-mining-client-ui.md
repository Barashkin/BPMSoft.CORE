# Email Mining Client UI

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: EmailEnrichmentMixin, EmailItemSchema, ContactEnrichmentSchema -->

> Клиентский слой показывает cloud icon, добавляет actions для обогащения
> контактов из письма и открывает enrichment window.

## EmailEnrichmentMixin

Файл: `Autogenerated/Src/EmailEnrichmentMixin.EmailMining.js`.

Миксин отвечает за:

- создание menu items;
- выбор caption для create/enrich contact;
- запрос `EnrichContactService`;
- показ/скрытие cloud indicator;
- публикацию sandbox message для окна enrichment.

Default service call:

```text
serviceName = EnrichContactService
methodName = GetCloudStateForEmail
data.activityId = current email activity Id
```

В action menu добавляются максимум первые 4 контакта из response.

## Email item integration

Файл: `Autogenerated/Src/EmailItemSchema.EmailMining.js`.

`EmailItemSchema`:

- подключает `EmailEnrichmentMixin`;
- задает `CallerSource = EmailPanel`;
- при `init` загружает enrichment actions;
- хранит `EnrchEmailData` и `EnrchEmailData.Status`;
- меняет icon action button на cloud icon;
- публикует `ContactEnrichmentPageVisibilityChanged`.

Actions загружаются, если:

```text
ReloadActions = true
или EnrchEmailData есть и EnrchEmailData.Status != ENRICHED
```

## Contact enrichment page

Файлы:

- `Autogenerated/Src/ContactEnrichmentSchema.EmailMining.js`;
- `Autogenerated/Src/ContactPageV2.EmailMining.js`;
- `Autogenerated/Src/ContactEnrichmentSchemaCSS.EmailMining.less`.

Страница enrichment получает:

- `contactId`;
- `contactName`;
- `enrchTextDataId`;
- `enrchEmailDataId`;
- `source`.

Затем показывает text data и позволяет сохранить выбранные значения через
`EnrichContactService.SaveEnrichedData`.

## Communication panel

Файлы:

- `Autogenerated/Src/CommunicationPanelEmailSchema.EmailMining.js`;
- `Autogenerated/Src/MainHeaderSchema.EmailMining.js`.

Email Mining встраивается в коммуникационную панель, чтобы пользователь видел
enrichment actions прямо в email item, а не только на карточке контакта.

## Client enums

Файл: `Autogenerated/Src/EmailMiningEnums.EmailMining.js`.

Используется для проверки статусов, включая `EnrichEmailDataStatus.ENRICHED`.
Для новых UI-проверок берите значения из enum, а не сравнивайте с literal
строками в нескольких местах.

## Типичные UI-проблемы

- Cloud icon не появился: проверьте `EnrchEmailData` и response
  `GetCloudStateForEmail`.
- Actions дублируются: `EmailItemSchema` проверяет `enrchTextDataId` первого
  элемента, поэтому при изменении формата response проверьте этот guard.
- Окно не открывается: проверьте sandbox message
  `ContactEnrichmentPageVisibilityChanged`.
- Кнопка есть, но данных нет: проверьте `GetTextDataForExistContact` или
  `GetTextDataForNewContact`.
