# Notifications Reminders Overview

<!-- Версия: 1.0 | Обновлено: 2026-04-27 | Платформа: BPMSoft 1.9 -->
<!-- Теги: Notifications, Reminding, NotificationV2, WebSocket, Push -->

> Входная точка в Notifications / Reminders Dive: модель `Reminding`,
> background jobs, серверная доставка, counters, UI-центр уведомлений и
> границы с ESN, Mobile и Activity.

## Границы

В этом dive описан NUI notification center:

- `Reminding` как базовая запись напоминания/уведомления;
- `NotificationEventListener`, `RemindingJob`, `NotificationCleanerJob`;
- `NotificationSender` и handlers доставки;
- `WebSocketNotificationSender` и `PushNotificationSender`;
- counters и `NotificationInfoService`;
- `BaseNotificationsSchema`, `ReminderNotificationsSchema`,
  `RemindingsModule`, `CenterNotificationModule`.

Не дублируются полностью:

- ESN notifications — см. [ESN notifications and mobile feed](esn-feed-notifications-mobile.md);
- mobile local/push UI — см. [Mobile services and push](mobile-services-push.md);
- Activity lifecycle fields — см. [Activity lifecycle](activity-lifecycle.md);
- Quartz API — см. [Scheduler Quartz](scheduler-quartz.md).

## Общий поток

```text
Source entity / process / provider
  -> Reminding
  -> RemindingJob
  -> RemindingRepository.GetRemindings
  -> NotificationSender
  -> WebSocketNotificationSender / PushNotificationSender
  -> ClientMessageBridge
  -> BaseNotificationsSchema / RemindingsModule / CenterNotificationModule
```
