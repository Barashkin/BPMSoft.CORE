## Константы статусов и типов (ActivityConsts)

```csharp
// ActivityConsts.Base.cs
public static readonly Guid NewStatusUId        = new Guid("384D4B84-58E6-DF11-971B-001D60E938C6");  // Новая
public static readonly Guid InProgressUId       = new Guid("394D4B84-58E6-DF11-971B-001D60E938C6");  // В работе
public static readonly Guid CompletedStatusUId = IntegrationConsts.ActivityCompletedStatusId;        // Завершена
public static readonly Guid CanceledStatusUId  = new Guid("201CFBA8-58E6-DF11-971B-001D60E938C6");  // Отменена
public static readonly Guid TaskTypeUId        = ...;  // Тип "Задача"
```

## Создание задачи

Минимальный набор: Title, StartDate, DueDate, Status, ActivityCategory, TypeId. Ответственный — OwnerId.

```csharp
// Пример из ActivityUserTask.ProcessDesigner.cs (ObsoleteCreateActivity)
var activity = new Activity(UserConnection);
activity.SetDefColumnValues();
activity.Id = Guid.NewGuid();
activity.TypeId = ActivityConsts.TaskTypeUId;
activity.Title = subject;
activity.StartDate = startDate;
activity.DueDate = NewDate(activity.StartDate, Duration, (ProcessDurationPeriod)DurationPeriod);
activity.ActivityCategoryId = ActivityCategory.IsEmpty() ? ActivityConsts.TaskCategoryId : ActivityCategory;
activity.ShowInScheduler = ShowInScheduler;

// Назначение ответственному
if (OwnerId != Guid.Empty) {
    activity.OwnerId = OwnerId;
    if (RemindBefore != 0) {
        activity.RemindToOwner = true;
        activity.RemindToOwnerDate = NewDate(activity.StartDate, -1 * RemindBefore, (ProcessDurationPeriod)RemindBeforePeriod);
    }
}
if (Account != Guid.Empty) activity.AccountId = Account;
if (Contact != Guid.Empty) activity.ContactId = Contact;

activity.ProcessElementId = processElementUId;
activity.AllowedResult = GetResultAllowedValues();
// Привязка к процессу и сохранение
UserConnection.IProcessEngine.AddProcessListener(activity, processElementUId, conditionData);
UserConnection.IProcessEngine.LinkProcessToEntity(Owner, activity.Schema.UId, activity.Id);
activity.Save();
```

## Обновление задачи (поля, статус)

```csharp
var schema = UserConnection.EntitySchemaManager.GetInstanceByName("Activity");
var activity = schema.CreateEntity(UserConnection);
if (!activity.FetchFromDB(activityId)) return;

// Изменение полей
activity.SetColumnValue("Title", "Новая тема");
activity.SetColumnValue("DueDate", DateTime.Now.AddDays(2));

// Смена статуса
activity.SetColumnValue("StatusId", ActivityConsts.InProgressUId);   // В работу
activity.SetColumnValue("StatusId", ActivityConsts.CompletedStatusUId); // Завершена
activity.SetColumnValue("StatusId", ActivityConsts.CanceledStatusUId);  // Отменена

// Назначение другому ответственному
activity.SetColumnValue("OwnerId", newOwnerContactId);
activity.Save();
```

## Назначение ответственному

```csharp
activity.SetColumnValue("OwnerId", contactId);
// Опционально: напоминание ответственному
activity.SetColumnValue("RemindToOwner", true);
activity.SetColumnValue("RemindToOwnerDate", dueDate.AddHours(-1));
activity.Save();
```

## Закрытие задачи (завершение / отмена)

**Завершение с результатом (из процесса):**

```csharp
// ActivityUserTask.ProcessDesigner.cs — CompleteExecuting
activity.SetColumnValue("ProcessElementId", null);
activity.UseAdminRights = false;
activity.Save();
// + синхронизация через SynchronizeCompletedUserTaskActivity при UseProcessPerformerAssignment
```

**Отмена (снятие слушателя процесса и статус «Отменена»):**

```csharp
// ActivityUserTask.ProcessDesigner.cs — CancelExecuting
UserConnection.IProcessEngine.RemoveActivityProcessListener(
    CurrentActivityId, processElementUId, ActivityConsts.CanceledStatusUId);
```

**Программное закрытие без процесса:**

```csharp
activity.SetColumnValue("StatusId", ActivityConsts.CompletedStatusUId);
activity.SetColumnValue("ResultId", resultLookupId); // при необходимости
activity.Save();
```
